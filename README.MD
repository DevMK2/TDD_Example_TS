다음과 같은 보고서가 있다고 하자.
|종목|주|가격|합계|
|:----:|:--:|:-----:|:--------:|
|IBM|1000|25|25000|
|GE|400|100|40000|
|-|-|합계|65000|

다중 통화를 지원하는 보고서를 만드려면 통화 단위를 추가해야 한다
|종목|주|가격|합계|
|:----:|:--:|:-----:|:--------:|
|IBM|1000|25 USD|25000 USD|
|GE|400|150 CHF|60000 CHF|
|-|-|합계|65000 USD|

또한 환율도 명시해야 한다
|기준|변환|환율|
|:--:|:--:|:--:|
|CHF|USD|1.5|
```
5 USD + 10 CHF = 10 USD 
```
